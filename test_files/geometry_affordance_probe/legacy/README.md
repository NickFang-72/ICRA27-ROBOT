# Legacy Geometry/Affordance Scripts

This folder keeps superseded experiment scripts out of the active workflow while
preserving the research history.

Use the active scripts in `../scripts/` and `../cair_setup_scripts/` for new
runs. Files here include old v1-v4 launchers, plan-guided experiments, one-off
review packet builders, older collectors, and initial single-view Qwen/RoboPoint
pilots.

These scripts are not maintained as the current entry points. Some reference
paths from their original locations, so treat them as history unless you first
port the paths back to the current layout.
